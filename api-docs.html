üîê Authentication APIs

POST /api/register
Register a new user account
email: string User email address
password: string User password
POST /api/login
Authenticate user login
email: string User email address
password: string User password
GET /api/me
Get current user information
Session: required Valid user session
POST /api/logout
Logout current user

üí≥ Payment APIs

POST /api/create-checkout-session
Create Stripe checkout session for payment
package_type: string 'full' or 'basic' package
payment_type: string 'card' or 'ach' payment method
GET /api/checkout/confirm
Confirm payment and unlock surveys
session_id: string Stripe session ID
POST /api/stripe-webhook
Handle Stripe webhook events (ACH payments)

üìä Data Management APIs (Admin Only)

Note: All admin endpoints require ?key=xie123 parameter

GET /api/admin/users?key=xie123
Get all users with survey completion status
GET /api/admin/users?key=xie123 Response: Array of user objects with completion flags
GET /api/admin/user-data/:email?format=csv&key=xie123
Download individual user data
email: string User email address
format: string 'csv' or 'json' (default: json)
GET /api/admin/user-data/test@gmail.com?format=csv&key=xie123 Response: CSV file download
POST /api/admin/bulk-download?key=xie123
Download multiple users' data
emails: array Array of email addresses
format: string 'csv' or 'json' (default: json)
POST /api/admin/bulk-download?key=xie123 Body: {"emails": ["user1@example.com", "user2@example.com"], "format": "csv"} Response: CSV file download
